Windows Server登入畫面中隱藏特定使用者

在Windows Server 2008開始，登入畫面慢慢的被修正成Windows 7的樣式，也就是你的電腦中如果有Administrator, John這兩位使用者，且都是啟動的狀態下，那麼登入畫面就會顯示這兩個大頭貼框框，等著你去點它一下，再輸入密碼。

在一般狀態下其實也不是問題，但是在特定的資訊安全考量下，你會倒想「為何我要在開始畫面提示看到這台電腦的操作者，這台伺服器有幾個帳號？以及完整的帳號名稱？」，如果你有想到這一點，那麼你可以用下列這個註冊檔稍微強化一下你的伺服器安全狀況。請開始＞執行＞regedit：

Windows Registry Editor Version 5.00
 
[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Winlogon\SpecialAccounts\UserList]
"你想隱藏的帳號名稱"=dword:00000000
