VMware關掉軟碟機（FloppyDisk）的方法

用VMware灌Windows Server後，在開機時都會啟動一台軟式磁碟機，無論你用怎樣的方法進去系統硬體設定都無法移掉，事實上正確的移除方法只要去BIOS關掉就好了，實在有一種無言的感覺。

To disable the floppy drive in the guest machine:

  1. Power off the virtual machine.
  2. Right-click the virtual machine and click Edit Settings.
  3. Click the Option Tab.
  4. Click Boot Options under Advanced.
  5. Select the The next time the virtual machine boots, force entry into the BIOS setup screen option.
  6. Power on the virtual machine.
  7. Using arrow keys, disable Legacy Diskette A:
  8. Press F10 to save the changes and exit. The machine reboots and completes the configuration.
