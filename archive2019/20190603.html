使用系統管理員身分執行cmd或powershell

其實藏了一堆快速鍵，把使用方式整理並記載在這裡。當然啦，所有的動作還是會觸發UAC（User Account Control）的。

方法一：Windows + R

這個方法是我最常用的方式，Windows + R，打入cmd或PowerShell後，同時按下「Ctrl + Shift + Enter」。

方法二：Windows + X

Windows + X後按下「A」即可。開啟cmd或PowerShell可以自己在設定裡面切換。

方法三：透過檔案總管

直接在檔案總管的URL列裡面，輸入cmd或PowerShell即可。這個方法可以直接進入到當前所在的目錄中，免再次CD更改目錄。但最大的缺點就是沒有快速鍵可以強制Run as Administrator。

方法四：透過檔案總管內建的選單

停留在某個目錄中，依照順序按下「Alt ＞ F ＞ S ＞ A」，就會自動幫你用PowerShell以最高管理者身分開啟至該目錄中，解決方法三的缺憾。

其實PowerShell可以解決大部分cmd中所有動作，如果你還是比較喜歡看到cmd，那可以利用下列指令切換：

已是系統管理員身分
－－－－－
cmd

非系統管理員身分
－－－－－
Start-Process cmd -verb runas
最高管理者 最高權限管理員 CMD Command PowerShell RunAsAdministrator HotKey ShortKey
