如何移除Windows Server 2016內建的Windows Defender Antivirus

Windows內建的防毒軟體Windows Defender Antivirus有他好用的地方，對於桌面用戶來說，可以省下一筆防毒軟體的費用，但對於伺服器版本（Windows Server 2016）的使用者來說，絕大多數的情景都是雞肋應用居多。

在早期的Windows Server 2016來說，要完整的關閉Windows Defender極其困難，把所有的工具，諸如：PowerShell、群組原則編輯器、服務管理員、伺服器管理員等都搬出來，最後仍然無法乾乾淨淨的移除關閉這項功能，幾經功能的迭代後，現在的移除已經改成非常的快速方便了，以下就來介紹移除的方法。

完整移除Windows Server 2016的Windows Defender Antivirus

Step 1. 打開伺服器管理員，點選管理＞移除角色及功能。

Step 2. 說明畫面，按下一步即可。

Step 3. 選擇要管理的伺服器。

Step 4. 伺服器的角色管理，這裡直接跳過。

Step 5. 來到伺服器功能區，請拉到下方找到Windows Defender功能，將裡面的兩個勾勾（Windows Defender、Windows Defender的GUI）都拿掉。

Step 6. 按下移除鈕，一切都移除完畢後，記得重開機以利伺服器正常作業。

大功告成！
