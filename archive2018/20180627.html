Windows 10開始選單之應用程式圖示（ICON）消失解法

Windows 10導入了敏捷式開發（Agile Software Development）後，歷經了多次的大版本迭代後，其實還是大大小小臭蟲不斷，到現在已經是1803版本了，但是連最基本的開始選單中，動態磚上應用程式的小圖示（ICON）竟然還是會有無故消失的問題，雖然說也不是大問題，但是看起來就是很惱人，今天特別找了個方法把它解決掉。

解決Windows 10開始選單中，動態磚上的圖示消失問題

Step 1. 首先，你可能會遇到長得像下圖的開始選單，選單中有很多圖示都自己不明的消失掉了，依據我的經驗，會這樣消失的電腦並不是每個人都會遇到，但一旦你有消失的問題發生，他就會開始偶爾的、隨機的消失幾個應用程式來惱怒你。

從上圖我們可以觀察到，無論你的應用程式是Win32還是UWP，總之都是會中獎的，所以這個BUG理論上與動態磚的更新無關。

Step 2. 請把下面的script儲存成「patch.bat」，放在你的桌面。

@echo off
taskkill /f /im explorer.exe
taskkill /f /im shellexperiencehost.exe
timeout /t 3 /NOBREAK > nul
del %localappdata%\Packages\Microsoft.Windows.ShellExperienceHost_cw5n1h2txyewy\TempState\* /q
timeout /t 2 /NOBREAK > nul
start explorer
@echo on

Step 3. 以系統管理員的身分，運行這隻「patch.bat」。

Step 4. 俗話說：「服用後到各大醫院檢查，就知效果。」
