逐步安裝Windows Server 2016評估版以及繁體中文語言包

因為Windows Server 2016官方釋出的評估版只有英文版本，或者是簡體中文版，但這並不符合台灣這邊的使用需求，因此這篇文章是在教你使用Windows Server 2016英文評估版（English - Evaluation），將其安裝起來並設定為繁體中文介面的教學文章。

下載Window Server 2016英文評估版

首先，請到微軟官方網站下載ISO檔案（Windows Server Evaluations），語言請選擇英文，然後請架設好自己的虛擬機器後，將ISO掛入開始安裝。

逐步安裝Window Server 2016英文評估版

Step 01. 語言選擇English（United State）、時間與地區格式選擇Chinese（Traditional, Taiwan）、鍵盤與輸入法選擇Microsoft Bopomofo（看你自己習慣甚麼樣的輸入法）。

Step 02. 點選Install Now。

Step 03. 選擇要安裝的Windows Server版本，在這邊我們選擇有桌面功能的版本（Windows Server 2016 Standard Evaluation with Desktop Experience）。如果選錯了，那麼重開機後只能使用PowerShell下指令喔！（Server Core Mode）

Step 04. 授權同意書當然是點選確定啦！

Step 05. 選擇自訂安裝。

Step 06. 接下來到了磁碟分割的階段，在這邊點選NEW。

Step 07. 將硬碟32GB的空間都切給單一Partition啦！

Step 08. Windows彈出警告，跟你講說會切一個他自己要施作的快速開機Partition。

Step 09. 這個快速開機的Partition竟然要500MB，可謂是逐代增長啊！

Step 10. 接下來經過一連串的安裝與重新開機後，就到了輸入Administrator密碼的時刻了。

Step 11. 按下完成後再跑一陣子，就可以看到經典的Windows登入畫面啦！

基本上Windows Server 2016的安裝就到此結束，接下來是進入後的繁體中文語言環境的設定。

安裝繁體中文語言包，並套用到整體伺服器環境

到Windows Server 2016這一代，基本上就已經沒有像Windows Server 2012那樣提供語言安裝包（LPK）讓人家直接下載檔案與安裝（至少我在網路上遍尋不），所以只能夠近系統裡面慢慢下載安裝啦！不過這也不算缺點，至少讓一般使用者可以透過更直覺的方式下載。

Step 12. 首先，我們先點選搜尋輸入（Control Panel），找到控制台選項，並點選進入。

Step 13. 在時間、語言與地區的分類裡面，點選「Add a language」。

Step 14. 點選中文（台灣）右側的Options選項。

Step 15. 點選Download and install language pack，就是從這邊直接連到微軟的官方網站下載繁體中文語言包啦！

Step 16. 等候他的更新作業，這邊「安裝」要等蠻久的時間，建議可以去喝杯茶或上個廁所。

Step 17. 裝好後回到原來控制台關於語言的地方，點選「Advance setting」進行進階設定。

Step 18. 在覆寫Windows全域顯示語言（Override for Windows display language）這個選項中，下拉選「中文（台灣）」，並且按下套用語言設定。（Apply language setting to the welcome screen, system accounts, and new user accounts）

Step 19. 彈出的地區選項確認視窗，點選OK。

Step 20. 套用完成，請點選Save後退出。

Step 21. 此時Windows會提示你說，這些套用語言設定的效果，必須要登出後再登入才會有效果。請點選現在就登出（Log off now）。

Step 22. 重新再登入後看一下系統視窗，YA，無論是開始選單、APP、還是所有的說明文字都變成繁體中文了，大功告成。

最後建議去設定，將Windows盡量更新到最新的修正，會比較妥當一些。所有的更新都安裝完成後，此刻的Windows Server 2016 Standard大約耗用17.5GB，以上數據提供給有用到的人做參考。
